# Weather-Based Wind Turbine Energy Prediction

## Description
Predicts wind turbine energy output based on weather using Machine Learning (Linear Regression, Random Forest, TensorFlow).  
Useful for energy forecasting, maintenance planning, and grid integration.

## Skills Used
Python, Data Preprocessing, Regression Algorithms, Numpy, Pandas, Matplotlib, Seaborn, TensorFlow, Scikit-Learn

## Project Structure
- `data/` - Dataset CSV
- `scripts/` - Preprocessing, Training, Prediction scripts
- `notebooks/` - EDA, Visualization, Model training
- `requirements.txt` - Python dependencies
